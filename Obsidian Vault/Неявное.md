[[Приведение типов]]
Неявные – выполняются автоматически (расширяющие преобразования, сужающие с потерей данных: int->float, long->float, long->double).

```java
public class Main{  
    public static void main (String[] args) {  
    byte b = 7;
	int d = b;
    System.out.println(d);  
    }  
}
```
